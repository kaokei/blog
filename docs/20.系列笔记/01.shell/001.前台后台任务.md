# linux 中的前台后台任务

## 执行后台任务

```Shell
nohup node any_command.js &
```

## 查询后台任务

```Shell
jobs
```

会输出如下格式的数据

```Shell
[1]  + running    nohup node any_command.js
```

注意到 jobs 只能显示当前 shell 对应的后台任务，如果你关闭了当前窗口，然后重新链接 ssh。此时 jobs 命令不会显示任何后台任务。

这时只能使用`ps -ef | grep 'something'`

## 把后台任务变成前台任务

```Shell
fg %1
```

## 直接杀死后台任务

```Shell
kill %1
```
